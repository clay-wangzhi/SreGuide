    Docker 设置代理

    起因：由于有些国外的镜像拉取不下来，所以设置翻墙代理，下载成功后，再导出docker save xx:tar -o xx.tar，传输到需要的节点导入docker load -i xx.tar

    ## 先停止Docker
systemctl stop docker

## 修改Docker配置文件
vim /lib/systemd/system/docker.service
## 在 [Service] 节点下面新增代理信息
[Service]
......
Environment="HTTP_PROXY=socks5://192.168.66.76:20080/" "HTTPS_PROXY=socks5://192.168.66.76:20080/"

## 重新加载Docker并启动 
systemctl daemon-reload
systemctl restart docker
